# Supplier AI Assistant

AI-сервис для поиска, оценки и выбора поставщиков по заданной категории товаров.
Проект разработан в рамках тестового задания на позицию AI Generalist.

---



## Описание

Сервис помогает специалистам по закупкам:

- находить поставщиков по категории товаров;
- сравнивать предложения;
- автоматически рассчитывать рейтинг поставщиков;
- получать рекомендации на основе AI-анализа;
- принимать решения быстрее и более объективно.

---

## Основные возможности

### Поиск поставщиков

Пользователь указывает:

- категорию товара;
- регион поиска.

Система подбирает подходящих поставщиков из базы данных.

---

### Автоматическая оценка поставщиков

Для каждого поставщика рассчитывается итоговый рейтинг (Score).

Оценка строится на основе нескольких факторов:

| Параметр | Вес |
|-----------|------|
| Рейтинг поставщика | 50% |
| Наличие сертификатов | 15% |
| Скорость доставки | 15% |
| Минимальный заказ | 10% |
| Ценовой уровень | 10% |

Максимальный итоговый балл — 100.

---

### AI-анализ

После выбора поставщиков пользователь может запустить AI-анализ.

Модель:

- анализирует найденных поставщиков;
- определяет наиболее подходящего кандидата;
- объясняет причины выбора;
- выделяет преимущества;
- указывает возможные риски.

## Запуск проекта

### 1. Клонирование репозитория

```bash
git clone <repository-url>
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Настройка OpenRouter

Создать файл:

```text
.env
```

Содержимое:

```env
OPENROUTER_API_KEY=your_api_key
```

---

### 4. Запуск приложения

```bash
streamlit run main.py
```
