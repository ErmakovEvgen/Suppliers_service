# Supplier AI Assistant

AI-сервис для поиска, оценки и выбора поставщиков по заданной категории товаров.
Проект разработан в рамках тестового задания.

---


## Описание

Сервис помогает специалистам по закупкам:

- находить поставщиков по категории товаров и региону;
- получить ключевую информацию по поставщику;
- сравнить предложения;
- автоматически рассчитать рейтинг поставщиков;
- получить рекомендации на основе AI-анализа;
- принять решение в выборе быстрее.


---

## Основные возможности

### Поиск поставщиков

Пользователь указывает категорию товара и регион поиска.

Система подбирает подходящих поставщиков из предоставленной базы данных.

---

### Автоматическая оценка поставщиков

Для каждого поставщика рассчитывается свой итоговый рейтинг.

Оценка строится на основе 6 факторов:

| Параметр | Вес |
|-----------|------|
| Рейтинг поставщика | 40% |
| Наличие сертификата | 15% |
| Сумма для бесплатной доставки | 10% |
| Минимальная сумма заказа | 10% |
| Уровень цен | 10% |
| Условия оплаты | 10% |

Максимальный итоговый балл — 100.

---

### AI-анализ

После поиска по категории или региону у пользователя есть возможность запустить AI-анализ для помощи с выбором подходящего поставщика по кнопке. Такой подход позволяет снизить количество запросов к модели, уменшить затраты и ускорить работу интерфейса.

Принцип работы AI-рекомендации:

- модель анализирует найденных поставщиков;
- определяет наиболее подходящего кандидата;
- объясняет причины выбора;
- выделяет преимущества;
- указывает возможные недостатки;
- подводит итог.

---

### Взаиодействие с результатом

После поиска у пользователя есть возможность переключаться между двумя режимами отображения поставщиков:

- Карточки
- Таблица

В режиме отображения "Карточки" поставщик набравших наибольшее количество баллов получает выделение на фоне конкурентов плашкой "🏆Рекомендуемый".
В режиме отображения "Таблица" у пользователя есть возможность сортировки, скрыть/показать выбранные столбцы, а тажке сохранить результат в формате CSV.

## Структура проекта

- /data/suppliers.json: - база данных с 50-ю поставщиками в разных категориях и регионах
- /services/supplier_service.py - функции поиска и расчёта итогового балла поставщика
- /services/llm_service.py - функция подключения AI-модели и промпт для её работы по генерации рекомендации выбора
- ui/components.py - Вывод информации о поставщиках в виде карточек
- ui/table.py - Вывод информации о поставщиках в виде таблицы с возможностью сортировки
- ui/info.py - Вывод информационной таблички с описанием подсчёта итогового рейтинга поставщика
- requirements.txt - Список зависимостей
- Конфигурация API-ключей хранится через переменные окружения.

## Логика работы

```text
Пользователь
        ↓
Ввод категории и региона
        ↓
Поиск поставщиков
        ↓
Расчёт рейтинга
        ↓
Сортировка результатов
        ↓
AI-анализ (по кнопке)
        ↓
Рекомендация подходящего поставщика
```

## Запуск проекта

### 1. Клонирование репозитория

```bash
git clone <repository-url>
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Настройка OpenRouter

Создать файл:

```text
.env
```

Содержимое:

```env
OPENROUTER_API_KEY=your_api_key
```

---

### 4. Запуск приложения

```bash
streamlit run main.py
```
